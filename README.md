# Pokemon-Wiki
A webpage that have searchable pokemon data and a mini game called "Guess the Pokemon".
Pokemon data is fetched using [PokeAPI](https://pokeapi.co/). 

Site is [live](https://sarcastic-soul.github.io/Pokemon-Wiki/).
